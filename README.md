# hsc-ui-vue2 组件库

### 快速开始

#### 1.安装组件库
```bash
npm i hsc-ui-vue2
```

#### 2.引用组件库
```javascript
// 全部引入
import 'hsc-ui-vue2/dist/css/index.css';
import HSCUI from 'hsc-ui-vue2';
Vue.use(HSCUI)

// 按需引入
import 'hsc-ui-vue2/dist/css/demo.css';
import { Demo } from 'hsc-ui-vue2';
Vue.use(Demo);
```
